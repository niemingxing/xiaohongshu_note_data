主要功能：

1.采集小红书推荐页笔记数据

2.采集小红书搜索页笔记数据

3.采集小红书搜索页用户数据

4.采集小红书博主页笔记数据

5.支持选择单篇笔记采集

6.同步到飞书多维表格
